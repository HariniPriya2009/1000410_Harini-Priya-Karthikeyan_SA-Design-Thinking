# 🌿 Smart Stress Relief App

### Design Thinking for Innovation – CRS Project

---

##  Project Overview

The **Smart Stress Relief App** is an AI-powered solution designed to reduce workplace stress through personalized relaxation experiences.
It integrates with **Smart Stress Relief Pods** to provide calming environments using music, lighting, aroma, and guided meditation.

The goal is to transform stress into **focus, calmness, and productivity** using design thinking principles.

---

##  Objective

* To understand user stress patterns through research
* To design a **personalized stress management system**
* To create an interactive prototype using design thinking
* To improve workplace well-being through AI-driven solutions

---

##  Design Thinking Process

### 1. Empathize & Define

* Conducted user surveys and research
* Identified key stress triggers:

  * Deadlines
  * Workload
  * Lack of breaks
* Created **user personas** representing different user types
* Developed empathy maps

 **Problem Statement:**
*How might we design an AI-powered system that provides instant, personalized stress relief without disrupting users’ daily workflow?*

---

### 2. Ideate

Brainstormed multiple solutions and categorized ideas into:

* Stress Tracking
* Relaxation Features
* AI Personalization
* User Experience

Final selected features:

* MoodSync AI
* Breathe with Me
* Pod Scheduler
* Stress Tracking Dashboard
* Post-Session Journal

---

### 3️. Prototype

A visual mobile app prototype was created using design tools.

###  Key Screens:

* Login / Signup
* Home Dashboard
* MoodSync AI Recommendations
* Pod Control Screen
* Guided Meditation
* Post-Session Journal
* Stress Tracking Dashboard

###  Core Features:

* **MoodSync AI:** Personalized stress-based recommendations
* **Breathe with Me:** Guided meditation sessions
* **Pod Scheduler:** Book relaxation sessions
* **Stress Tracking Dashboard:** Monitor stress levels
* **Post-Session Journal:** Reflect on experience

---

### 4️. Test

Testing was conducted with multiple users to evaluate usability and effectiveness.

####  Testing Method:

* Users interacted with the prototype
* Observed navigation and behavior
* Collected feedback using rating scales

####  Feedback Summary:

* Easy to use interface
* Effective stress relief features
* AI recommendations were helpful

####  Issues Identified:

* Navigation clarity needed improvement
* More personalization options required

####  Improvements Made:

* Enhanced UI clarity
* Improved AI suggestions
* Simplified navigation flow
